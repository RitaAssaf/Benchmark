# Benchmark
This benchmark contains graphs used in subgraph isomorphism for position graphs. Position graphs are with specific strucure that can be modeled in rows/cols graphs. We have classes of 40, 200 and 600 nodes.

The repository was remotely atteched to the folder /home/etud/Bureau/projet/benchmark that contains position graphs of 3 classes only.

If one wishes to generate rows/cols graphs, he must use rows_columns_generator script.
